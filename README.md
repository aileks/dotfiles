# Dotfiles

Ubuntu 24.04+ dotfiles with the [Ashen](https://codeberg.org/ficd/ashen) colorscheme.

## Quick Start

```bash
curl -fsSL https://aileks.dev/linux | bash
```

Or locally:

```bash
git clone https://codeberg.org/aileks/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```
