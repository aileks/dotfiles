# Arch Linux Dotfiles

Arch setup built on Niri with the [Ashen](https://codeberg.org/ficd/ashen) colorscheme.

## Quick Start

```bash
curl -fsSL https://aileks.dev/dotfiles | bash
```

Or manually:

```bash
git clone https://codeberg.org/aileks/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./setup.sh
```
