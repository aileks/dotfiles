# Addicted to Configuration

### Dotfiles for pop!_os alpha

#### Includes configuration for:

- Zsh
- Neovim
- Tmux
- Fastfetch

#### Install general programs:

[fastfetch](https://github.com/fastfetch-cli/fastfetch) needs a 3rd party repo
```sh
sudo apt install git neovim zsh tmux fzf zoxide ripgrep eza trash-cli fastfetch dunst
```
Note: I use a [paid font](https://berkeleygraphics.com/typefaces/berkeley-mono/) for some of my configuration.
