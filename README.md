# Fedora 42 Setup

> [!IMPORTANT]
> Some of these configurations use [Berkeley Mono](https://usgraphics.com/products/berkeley-mono), which is a paid font. You may need to purchase it or replace it with a font of your choice in the configuration files.


## Usage

```sh
git clone https://github.com/aileks/dotfiles.git
cd dotfiles
./install.sh
```
